Admissions Connect Autofill

About

The Berea College Connect Autofill extension for Firefox and Chrome was created to improve various processes within Admissions and reduce data entry errors. Currently it is maintained by Jacob Patton in the Office of Admissions. For questions or comments, please email jacob_patton@berea.edu

The functions below are based on the view in Connect and use watch for certain attributes to be available. Renaming a view or removing fields from the view may negatively impact the extension and cause parts of it not to work. This should show an error alert box. However, if a view is slow to load, this may also trigger the error alert box.

Functions

Below is a list of everything that the extension currently does.

Campus Visit Itinerary
- Creates a button that allows the visit type (morning, mid-day, or afternoon) and activities to be automatically populated or cleared. (Tweaked for 2018 Summer)
Campus Visit Reservation
- When the "Visit Date" is set, the correct "Visit Date" is selected. Clicking on the field will ask if you want to clear it, which would also clear the "Visit Day". Must use the date picker to set the date.
- When "Visit Type" is selected, the correct "Arrival Time" is populated or cleared.
HS Transcript / Counselor Evaluation Form
- Data that is entered into the "High School Name" field, this is automatically. copied to the "High School Transcript Name" field. While "High School Transcript Name" can be updated without changing "High School Name", any changes to "High School Name" will overwrite these changes.
- When the "Class Rank", "Class Size", or "Real Percentil Req?" is updated, the "Class Rank Percentile" is calculated and "Percentile Rec'd Date" is set for current date. (Note: If Percentile Required is flipped to "No", it removes the current percentile.)
- If "GED %" is entered and the "High School Transcript Name" is set to "GED", the "Class Rank", "Class Size", "Class Rank Percentile" are cleared, "Percentile Req?" is set to "No", and the date is set for the current day.
Evaluations and Endorsements
- As soon as data is entered into "Real Evaluation-Endorsement 1" or "Real Evaluation-Endorsement 2", the evaluation date is populated. As long as the date field is not empty, it will not repopulate.
- The data entered into the "Real Evaluation-Endorsement 1" or "Real Evaluation-Endorsement 2" is copied to the shadow attribute, which is made read only.
Dec of Intent / Ent Fee / Non-Enrolling Form
- When the "Fall Ent Fee" date is populated, the "Fall Ent Fee" amount is populated with "50" to represent the $50 deposit. Clicking on the populated field will ask if you want to clear it, which woudl also clear the "Fall Ent Fee" amount. Must use the date picker to set the date.
Interview Score Summary-Form
- When the interview total is entered, if the total score is provided does not match the total of adding the questions together, the background of the field is colored red. If they match, it is colored green.
International Apps Processing
- When the "Intl App Status Indicator" is set for the first time, the "Intl App Processing Date" is populated with the current date.
